### 2. Surgical Robot Trajectory Planning

**Classical Control + Deep RL Foundation**

- Velocity-level inverse kinematics with Jacobian mapping
- Multiple PID tuning methods compared (Ziegler-Nichols, Bayesian, PSO)
- Foundation for RL-based adaptive trajectory optimization
- Ready for hardware transfer

**Technologies:** Python, MuJoCo, Inverse Kinematics, PID Control, Cubic Splines
